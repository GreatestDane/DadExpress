HELP FILE FOR:   PROGRAM FOR SOLVING A RIGHT TRIANGLE

 

DEFINITION:

 

A “right triangle” is defined as one where two of its legs have a 90-degree angle between them.  This makes them square with each other.  The sum of the other two angles of the triangle equal 90 degrees.  Refer to the right triangle sketch.

 

PROGRAM OBJECTIVE:

 

When the user enters values for any two components of the right triangle, the program calculates values for the other two components.  Zero values must be entered for the unknown components.  

 

Refer to the example at the end of this help file.

 

UNITS:

 

            Input can be in either feet or meters.  Output will be in the same units as the input.  The program has been written such that unit conversions are not necessary.  Enter the input in the units desired and click the radio button for these units.  This will ensure that the printout will show what units were actually used in the calculations for later reference.   

 

ANGLE THETA:

 

This angle is always common to the hypotenuse and the side adjacent.  The angle between the hypotenuse and the side opposite is not listed for clarity.  Its value is equal to (90 – theta) in degrees.

 

HYPOTENUSE:

 

            This is the longest leg of a right triangle.  It is always on a slope relative to the other two legs.

 

SIDE ADJACENT:

 

The side or leg of the triangle that is next to the origin.

 

SIDE OPPOSITE:

 

            No comment.

 

EXAMPLE 1:

 

The “sample values” button at the bottom of the screen contains input for a 3, 4, 5 triangle.  This is the triangle that is used by carpenters to check two adjoining walls to ensure that they are square with one another.  By clicking on “sample values” the input values are shown with the output values blank.  By then clicking on “calculate”, both the input and output values are shown.  The units are in FEET.
